# أرشيف المستندات - Document Archiver

تطبيق ويب لأرشفة وإدارة المستندات باستخدام Next.js و LowDB.

## الميزات

- ✅ إضافة وتحرير وحذف المستندات
- ✅ تصنيف المستندات بالفئات
- ✅ إضافة وسوم للمستندات
- ✅ البحث في المستندات
- ✅ فلترة المستندات بالفئة
- ✅ واجهة مستخدم باللغة العربية
- ✅ تخزين البيانات محليًا باستخدام LowDB
- ✅ تصميم متجاوب يعمل على الهاتف والحاسوب

## التقنيات المستخدمة

- **Frontend**: Next.js 14, React, TypeScript
- **Styling**: Tailwind CSS
- **Database**: LowDB (JSON-based)
- **Validation**: Zod
- **Language**: Arabic (RTL Support)

## متطلبات التشغيل

- Node.js 18+ 
- npm أو yarn

## التثبيت والتشغيل

### 1. تثبيت التبعيات
```bash
npm install
```

### 2. تشغيل التطبيق في وضع التطوير
```bash
pip uninstall greenlet
pip install --no-cache-dir greenlet
```

سيعمل التطبيق على: http://localhost:3000

### 3. بناء التطبيق للإنتاج
```bash
npm run build
```

### 4. تشغيل التطبيق في وضع الإنتاج
```bash
npm start
```

## استخدام التطبيق

### إضافة مستند جديد
1. انقر على زر "إضافة مستند جديد"
2. املأ البيانات المطلوبة:
   - العنوان (مطلوب)
   - المحتوى (مطلوب)
   - الفئة (اختياري)
   - الوسوم (اختياري)
3. انقر على "حفظ المستند"

### البحث في المستندات
- استخدم مربع البحث للبحث في العناوين والمحتوى والوسوم
- اختر فئة معينة من القائمة المنسدلة للفلترة

### إدارة المستندات
- يمكن عرض تفاصيل كل مستند
- تاريخ الإنشاء والتحديث يظهر تلقائيًا
- الوسوم تظهر كعلامات ملونة

## قاعدة البيانات

يستخدم التطبيق LowDB لتخزين البيانات في ملف JSON محلي. البيانات تشمل:

- **المستندات**: العنوان، المحتوى، الفئة، الوسوم، تواريخ الإنشاء والتحديث
- **الفئات**: عام، فواتير، عقود، مستندات شخصية
- **الوسوم**: يتم إنشاؤها تلقائيًا من المستندات

## الأمان والخصوصية

- جميع البيانات تُخزن محليًا على الخادم
- لا يتم إرسال البيانات لخوادم خارجية
- يمكن إضافة نظام مصادقة في المستقبل

لإعادة بناء .exe: npm run dist.
